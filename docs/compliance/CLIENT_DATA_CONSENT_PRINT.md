## 二、数据使用说明

为提升诊疗质量、复盘临床流程、开发院内临床辅助工具，我院可能在脱敏 / 去标识化后，将本次就诊产生的部分资料用于院内质量改进、模型训练、算法评估和临床研究。

## 三、可能使用的数据类型

请勾选：

```txt
[ ] 影像资料：X 光 / DR / CT / MRI / 超声 / 内窥镜等
[ ] 检验结果：血常规、生化、电解质、尿检、血气等
[ ] 电子病历文本：主诉、体检、诊断、用药、出院记录等
[ ] 临床照片 / 术中视频
[ ] 诊断标签与临床操作记录
```

## 四、脱敏方式

我院会尽量去除或替换以下直接身份信息：

```txt
主人姓名
主人电话
证件号
详细地址
可直接识别身份的病历编号
影像文件头中的可识别字段
```

## 五、不同意与撤回

```txt
您可以不同意，且不影响诊疗服务。
您可以随时撤回同意。
撤回后，我院将在 30 日内停止将新数据用于训练或导出；
90 日内将此前可移除数据标记为不可用。
已经进入模型训练的权重通常无法按个体完全回溯删除，但您的数据会从后续训练与导出中排除。
```
