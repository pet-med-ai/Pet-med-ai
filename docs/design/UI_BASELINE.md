# Pet-Med-AI UI 定型参考

## 定型界面

本文件记录用户确认喜欢并希望后续作为定型方向的两张界面参考图：

1. 登录页参考：`pet-med-ai-ui-login-reference.jpeg`
2. 首页 / 工作台参考：`pet-med-ai-ui-home-reference.jpeg`

## 总体定位

Pet-Med-AI 是宠物 AI 辅助诊断系统，界面风格定为：深色科技医疗风、AI 智能问诊、犬猫 + 异宠兼容、多模块临床工作台。

## 视觉基调

- 背景：深蓝黑、暗色星网、科技网格、医疗 AI 氛围。
- 主色：霓虹蓝 / 青蓝。
- 辅色：紫色，用于异宠、强调按钮、分组标签。
- 点缀：绿色用于在线、通过、安全；橙色 / 红色用于风险和预警。
- 质感：玻璃拟态卡片、发光边框、渐变按钮、轻量发光图标。
- 不采用：白底传统后台、普通 CRUD 管理页、卡通幼稚风、廉价模板风。

## 登录页定型要点

- 左侧为品牌与能力展示：Pet-Med-AI、宠物 AI 辅助系统、犬猫 + 异宠兼容。
- 左侧核心视觉：犬、猫、兔、鸟、爬宠、啮齿类同台展示，体现多物种支持。
- 登录卡片置于右侧，深色玻璃卡片风格。
- 登录项包含：账号 / 手机号 / 邮箱、密码、登录身份、记住我、忘记密码。
- 主按钮为高亮蓝色发光按钮。
- 次级按钮为紫色描边注册 / 申请试用、游客预览。
- 底部强调数据安全、加密传输、信息安全。

## 首页 / 工作台定型要点

- 左侧固定导航：首页、AI 咨询、病例管理、影像中心、检验报告、知识库、药品与治疗、客户与随访、数据分析、系统设置。
- 顶部有品牌、搜索、通知、医生身份信息。
- 顶部物种切换：犬猫主入口 + 异宠入口，并列展示犬、猫、兔、鸟类、爬宠、啮齿类等。
- 主区域三栏布局：
  - 左：AI 智能问诊 / 分诊对话。
  - 中：病例摘要卡、风险等级、鉴别诊断 Top 3、建议检查、病史摘要。
  - 右：AI 快速洞察、风险预警、分诊级别、下一步建议、影像 / 检验上传区、今日数据概览。
- 底部有多物种知识库 / 适配指南卡片。

## 产品信息架构方向

后续前端应逐步从当前联调面板升级为正式工作台：

1. 登录页正式化。
2. 首页 / 工作台正式化。
3. AI 问诊模块卡片化。
4. 病例摘要、风险、鉴别诊断、建议检查结构化展示。
5. 犬猫与异宠通过物种导航进入不同规则库和问诊模板。
6. 影像 / 检验上传模块作为后续阶段接入。
7. 知识库入口保留，为 JSON/YAML 规则库和未来后台管理做铺垫。
