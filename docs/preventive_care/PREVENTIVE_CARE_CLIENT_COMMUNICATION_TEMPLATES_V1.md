# Preventive Care Client Communication Templates V1

## Purpose

Draft language for future manual contact.

These are not sent automatically in V1.

## In-app reminder draft

```txt
【预防保健提醒】{pet_name} 可能已到 {reminder_type} 的复核时间。请联系医院确认具体接种/驱虫方案。此提醒不替代兽医诊断和处方。
```

## Front desk phone script

```txt
您好，这里是瀚森宠物医院。系统显示 {pet_name} 的 {reminder_type} 需要复核，我们想帮您预约医生确认具体时间。请问近期是否方便到院？
```

## Opt-out acknowledgement

```txt
已记录您暂不接收此类提醒。该设置不影响正常诊疗服务；如需恢复提醒，可联系前台。
```

## Required review

Before outbound use:

```txt
legal/compliance review
clinic owner approval
client consent support
opt-out support
send log support
```
